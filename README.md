# fox-step-privacy

Privacy policy page for the [Fox Step](https://github.com/ReinaryuIlyen/fox-step) Android app.

Served via GitHub Pages at <https://reinaryulyen.github.io/fox-step-privacy/>.

The URL is referenced from the Fox Step listing on Google Play Console.
